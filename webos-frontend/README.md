# WebOS Frontend

基于 React + Vite + shadcn/ui 的文件管理前端界面。

## 技术栈

- React 18
- Vite 5
- TypeScript
- shadcn/ui
- Monaco Editor (代码编辑器)
- Tailwind CSS
- Axios

## 安装

```bash
pnpm install
```

## 开发

```bash
pnpm dev
```

访问 http://localhost:3000

## 构建

```bash
pnpm build
```

## 预览构建结果

```bash
pnpm preview
```

## 功能

- 文件/文件夹浏览
- Monaco Editor 在线编辑
- 文件上传/下载
- 创建文件夹
- 重命名/删除文件
- 面包屑导航
- 历史记录（前进/后退）
- 响应式设计
